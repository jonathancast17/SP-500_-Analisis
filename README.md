# SP-500_-Analisis
Analisis de Crashes historicos del S&amp;P500 con Python

Análisis de caídas históricas del mercado americano 
usando datos reales de Yahoo Finance.

## ¿Qué hace?
- Descarga datos históricos del S&P 500 desde 2020
- Visualiza precio histórico y cambios diarios
- Identifica los 10 peores días del mercado

#Tecnologías
- Python 3
- yfinance
- matplotlib
- numpy

#Resultados
Los 10 peores días desde 2020 incluyen principalmente 
el crash de COVID (marzo 2020) y volatilidad reciente 
de abril 2025 por tensiones comerciales.
